# Atlas

`atlas` 是这个仓库的全局视野层。它不承载长篇学习笔记，而是回答三个问题：

- 这棵 infra 树有哪些节点？
- 每个节点现在应该排什么优先级？
- 每个叶子节点如何证明自己真正学会了？

建议在 Obsidian 中把这里作为入口页，然后把每个叶子学习项链接到对应目录下的笔记或实验。
